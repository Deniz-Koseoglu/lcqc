### Отчётность

Этот модуль используется для экспорта кратких отчётов об эффективности ВЭЖХ-колонок в формате .PDF, служащих **сертификатом обеспечения качества**. Предоставленные опции в основном понятны и перечислены ниже.

#### Параметры отчёта
Эта вкладка позволяет пользователю выбрать, какие данные о пиках включить в отчёт, с возможностью присвоения названий и концентраций.

**Выбор диаграммы**
: Выберите источник хроматограммы для отображения в отчёте. При выборе **Интегрирование пиков** или **Деконволюция пиков** данные берутся из модулей **5. Интегрирование** или **6. Деконволюция** соответственно.

**Введите индексы пиков**
: Позволяет пользователю вводить **индексы** пиков для включения в отчёт, разделённые либо **двоеточием** (:), либо **запятыми** (,). Интерактивный селектор пиков также доступен в **Визуальном выборе**.

**Введите названия пиков**
: Облегчает ввод названий пиков (**необязательно**). Также доступно в **Визуальном выборе**.

**Введите концентрации пиков**
: Облегчает ввод концентраций пиков (**необязательно**). Также доступно в **Визуальном выборе**.

**Единицы концентрации**
: Используется для указания текстовой строки в качестве единицы концентрации (например, **ppm**).
<br></br>
#### Теоретические тарелки
Пользователь может выбрать, какие метрики теоретических тарелок включить в отчёт в этом разделе. В настоящее время могут быть включены метрики из 4 различных методов. Это: **Европейская фармакопея (EP)**, **Площадь-Высота (AH)**, **5-Сигма (S5)** и/или **Экспоненциально-модифицированная гауссова (EMG)**. Данные берутся из модуля **8. Метрики эффективности**.
<br></br>
#### Метрики асимметрии
Аналогично предыдущему разделу, здесь выбираются метрики асимметрии для включения, такие как **Коэффициент асимметрии (As)**, **Коэффициент хвостования USP (Tf)** и/или графики **полного анализа пиков (TPA)** (если доступны). Обратите внимание, что для последних в настоящее время поддерживается до **6 графиков**. Данные берутся из модуля **7. Симметрия пиков**.
<br></br>
#### Метрики, специфичные для колонки
Флажок позволяет дополнительно включить специфичные для колонки дополнительные метрики из модуля **8. Метрики эффективности** (если доступны).
<br></br>
#### Содержание отчёта
Этот раздел позволяет пользователю выбрать статическую информацию для отображения в отчёте, такую как серийный номер колонки, подвижная фаза, номер документа, имя оператора контроля качества и т.д. Предоставлены примеры-заполнители.
<br></br>
#### Параметры вывода
В этом разделе может быть загружен пользовательский логотип компании (в формате **.PNG** или **.JPG**) для отображения в заголовке документа. Здесь также можно выбрать соотношение сторон хроматограммы. Наконец, отчёт может быть экспортирован после выбора **существующего** каталога вывода.
