### Коррекция базовой линии

Этот модуль выполняет коррекцию базовой линии импортированной хроматограммы с использованием различных алгоритмов. 
Вкладка **Основные** содержит выпадающий список для выбора между различными методами коррекции:

**Итеративный асимметричный метод наименьших квадратов (ALS)**
: Итеративная сходимость на основе взвешенного сглаживания Эйлерса (т.е. Уиттакера-Хендерсона) со штрафом конечных 
разностей порядка 2 (Eilers, [2003](https://doi.org/10.1021/ac034173t)). Подробное объяснение и примеры см., например, 
в работе Peng *et al.* ([2010](https://www.doi.org/10.1016/j.aca.2010.08.033)).

**Метод Чанга**
: Робастный алгоритм для хроматограмм и спектров с высокой плотностью сигналов, разработанный Chang *et al.* 
([2007](https://doi.org/10.1016/j.jmr.2007.05.008)).

**Итеративные сглаживающие сплайны с корректировкой корневой ошибки (ISREA)**
: Использует сглаживающие сплайны для оценки базовой линии (Xue *et al.*, [2021](https://www.doi.org/10.1177/0003702820955245)).

**Модифицированная полиномиальная аппроксимация (ModPolyFit)**
: Итеративный алгоритм коррекции базовой линии на основе автоматической полиномиальной аппроксимации, разработанный 
Lieber & Mahadevan-Jansen ([2003](https://www.doi.org/10.1366/000370203322554518)).

Коррекцию базовой линии также можно полностью пропустить, выбрав **Нет** (по умолчанию).

#### Дополнительные параметры

Вкладка **Дополнительно** изменяется контекстно в зависимости от выбранного метода коррекции и включает различные 
параметры, специфичные для метода:

1. Параметры **ALS** включают ограничение второй производной (**Lambda**), веса положительных остатков (**p**), 
допустимую погрешность (точность), необходимую для достижения сходимости (**prec**), и максимальное количество 
итераций (**Макс. итераций**).

2. Параметры **метода Чанга** включают положение базовой линии относительно шумовой компоненты (**Порог**), 
параметр фильтра высоких частот (**alpha**), долю низкоинтенсивных фрагментов отфильтрованного сигнала, которые 
считаются базовой линией (**Доля базовой линии**), количество сегментов для разделения отфильтрованного сигнала 
(**Сегменты**), размер окна сигнала в точках (**Окно сигнала**) и метод аппроксимации базовой линии 
(линейный или кубический; **Метод аппроксимации**).

3. Параметры **ISREA** включают критерий сходимости от **0.0001** до **10** (**Eta**) и максимальное количество 
итераций (**Макс. итераций**).

4. Параметры **ModPolyFit** включают степень полиномиальной аппроксимации (**Степень**), точность, необходимую 
для сходимости (**Точность**), и максимальное количество итераций (**Макс. итераций**).

Все методы также включают возможность удаления отрицательных значений из скорректированного сигнала 
(**Удалить отрицательные**), чтобы значения базовой линии никогда не превышали значения исходного сигнала.
